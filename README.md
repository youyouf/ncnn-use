# NCNN
AI inference acceleration demo as NCNN

## ncnn简介

ncnn 是一个为移动端和嵌入式设备优化的高性能神经网络前向计算框架。它具有轻量级、高性能和跨平台等特点，适用于各种资源受限的设备。
